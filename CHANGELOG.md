# Changelog

All notable changes to this project will be documented in this file.

## [Released]

## [1.0.0]

- First release.

- Make stable release.

- Added keyboard event `keyup` for update value ([#5](https://github.com/kenhyuwa/litepie-datepicker/pull/5)).
